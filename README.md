# Описание работы проекта

Этот проект представляет собой веб-приложение на базе Node.js, подготовленное к контейнеризации.

### Как это работает:

1.  **Серверная часть (`server.js`)**: 
    - Использует Node.js и Express (согласно `package.json`) для обработки входящих HTTP-запросов.
    - Служит API-прослойкой и раздает статический контент из папки `public/`.

2.  **Клиентская часть (`public/index.html`)**:
    - Отображает пользовательский интерфейс в браузере.

3.  **Конфигурация (`.env.example`)**:
    - Содержит шаблоны настроек (порты, ключи API), которые необходимо скопировать в файл `.env` для локальной работы.

4.  **Развертывание (`Dockerfile`)**:
    - Автоматизирует создание окружения. При сборке устанавливаются зависимости из `package.json`, копируется исходный код и запускается сервер.

### Порядок запуска:
1. Создайте `.env` на основе `.env.example`.
2. Выполните `npm install` для установки зависимостей.
3. Запустите проект командой `node server.js` или соберите Docker-образ: `docker build -t nexus-app .`.
