# Веб-панель Nexus Orchestrator

Этот проект представляет собой панель управления в реальном времени.

## Как запустить:
1. Зайдите в **Settings** вашего репозитория на GitHub.
2. Перейдите во вкладку **Pages**.
3. В разделе **Build and deployment** выберите ветку `main` и папку `/ (root)`.
4. Нажмите **Save**. Через минуту панель будет доступна по ссылке, которую предоставит GitHub.

## Функционал:
- Мониторинг ресурсов.
- Управление командами.
- Логирование действий.
